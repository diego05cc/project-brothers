# CODEX_START_HERE.md ## Proyecto **Nombre provisional:** Barrio Digital NYC Este proyecto es una microagencia bilingüe enfocada en ayudar a pequeños negocios locales, especialmente negocios latinos de Queens y Nueva York, a conseguir más clientes mediante: - Sitios web optimizados para celular. - Botones directos de llamada y WhatsApp. - Mejor presentación de servicios, menús y precios. - Optimización básica de Google Business. - Contenido para Instagram y Google. - Sistemas sencillos para solicitar reseñas reales. - Mantenimiento mensual. El propietario del proyecto está comenzando desde cero y necesita que Codex actúe como desarrollador principal, explicando cada paso de forma sencilla. --- ## Objetivo principal Convertir este prototipo en un negocio real, empezando por: 1. Completar y personalizar la página de ventas de Barrio Digital NYC. 2. Usar Patojitos Store como primer caso de demostración. 3. Publicar ambas páginas en internet. 4. Crear un proceso repetible para producir páginas para nuevos clientes. 5. Conseguir los primeros tres clientes pagos. --- ## Archivos principales - `index.html` - Página principal de ventas para Barrio Digital NYC. - Presenta servicios, paquetes, precios y contacto. - `patojitos-demo.html` - Página de demostración para Patojitos Store. - Debe servir como ejemplo real del tipo de trabajo que la agencia vende. - `VENTAS.md` - Guiones para contactar negocios. - Auditorías gratuitas. - Manejo de objeciones. - Rutina diaria de prospección. - `ENTREGA.md` - Proceso para recibir información, construir, revisar y entregar proyectos. - `FORMULARIO_CLIENTE.md` - Preguntas para recopilar información de cada cliente. - `README.md` - Explicación general del kit. --- ## Primera misión de Codex Revisar todo el proyecto y preparar una versión lista para publicar. ### Tareas prioritarias 1. No debe parecer una plantilla genérica de inteligencia artificial. Mantener una combinación visual cálida, con fondos claros, contraste fuerte y llamados a la acción visibles. --- ## Diseño de Patojitos Store La página de Patojitos Store debe sentirse: - Tierna. - Rosada. - Maternal. - Profesional. - Protectora. - Casera. - Fácil de usar desde un teléfono. El objetivo principal es que el visitante pueda: 1. Ver el menú. 2. Ver precios. 3. Conocer la zona de entrega. 4. Escribir por WhatsApp. 5. Entender horarios y métodos de pago. No agregar funciones complejas durante la primera versión. --- ## Paquetes iniciales del negocio ### Arranque — $349 - Página de una sola sección larga. - Diseño para celular. - Botones de llamada y WhatsApp. - Español o inglés. - Una ronda de cambios. ### Crecimiento — $699 - Hasta cinco secciones. - Español e inglés. - Revisión básica de Google Business. - Doce textos para redes. - Código QR para solicitar reseñas. - Dos rondas de cambios. ### Mantenimiento — $149 al mes - Cambios pequeños. - Cuatro publicaciones mensuales. - Borradores para responder reseñas. - Reporte básico. - Soporte por mensaje.
